# Cri
The data set was acquired using the Nuance Cri hyperspectral sensor. It images several rocks in a grassy area. The spatial size is 400Ã—400 pixels. The spectral resolution is 10 nm and 46 bands are reserved. 
# Objects to be Detected in Cri
There are ten stones in different locations identified as objects of interest to be detected.
